Attributes and links

Attributes

The most commonly used attributes are title, lang, spellcheck, and tabindex.

title: Specifies extra info on hover.

lang: Specifies the language the element is in.

spellcheck: Turn on or off spellcheck.

tabindex: Specifies the order to cycle elements with the tab key.

This is a new paragraph, and it has been made to test the working of title attribute.

Links

The link tag is used with an href attribute, which links to a URL to the text inside the link tag.

The value of the target attribute decides whether the URL is opened in the same tab, a new tab of a specifc frame.
